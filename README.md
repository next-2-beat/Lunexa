# Lunexa Music Bot

Lunexa is a modern, powerful Discord music bot designed for **smooth playback, high performance, and full automation**.

Built to handle **large Discord servers** with stability, speed, and crystal-clear audio quality.

---

## Why Lunexa?

Lunexa is not just another music bot — it’s a **complete music system** for Discord.

- Lag-free music playback  
- Lightning-fast response time  
- 24/7 stable uptime  
- Clean, simple, and intuitive commands  

---

## Features

- High-quality audio streaming
- Supports **YouTube, Spotify, SoundCloud**, and more
- Fast & stable Lavalink playback
- Advanced queue management system
- Loop & shuffle modes
- Playlist support
- Slash command support
- Auto-reconnect & error handling
- DJ & Admin permission system
- Multi-server (multi-guild) support

---

## Music Controls

- Play / Pause / Resume
- Skip / Stop
- Volume control
- Loop (Track / Queue)
- Shuffle queue
- Playlist creation & management

---

## Database

Lunexa uses **MongoDB** to store:

- Guild (server) configurations
- User preferences
- Music & player settings
- Premium / feature flags (optional)

---

## Security

- Environment-based configuration
- Secure Discord token handling
- MongoDB authentication
- Crash protection & error recovery
- Clean, modular, and scalable architecture

---

## Tech Stack

- **Node.js**
- **Discord.js**
- **MongoDB**
- **Lavalink**
- REST & WebSocket APIs
